# Animal Kingdom Database

A web app that users can search about animals and find out more about their classifications such as phylum, class, order and more. 

## Stack
- Next.js
- Tailwind.css
- Shadcn
- Prisma
- GraphQL

## Setting Up Locally

Prerequisites
- Postgresql 16+
- Node.js 18
